Description

calculate the network of one pathway

Usage

network_seq_correlation(
  pathway,
  seuratObj,
  genes_selected,
  type1,
  type2,
  returnType = "network"
)

Arguments

pathway

seuratObj

A Seurat Oject, with status in meta.data

genes_selected

Gene selected to be analyzed

type1

the first type (defined in status) for analysis

type2

the second type (defined in status) for analysis

list
